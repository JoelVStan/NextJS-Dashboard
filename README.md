<h2>🛠️ Installation Steps:</h2>

<p>1. Install packages</p>

```
npm install
```

<p>2. Run in dev mode</p>

```
npm run dev
```

<p>3. Build the project</p>

```
npm run build
```

<p>4. Start the project</p>

```
nm start
```
